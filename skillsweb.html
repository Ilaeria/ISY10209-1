Skills: Web Development

This website is an example of my web development skills. It was written by hand in a basic text editor (Notepad++) and all HTML and CSS is my own. My logo was created entirely with CSS.

Colours, fonts, and layouts were chosen to be as "web safe" as possible. This aims to ensure the website displays on different monitors consistently.

A print-friendly style sheet is linked to every page on this website, so that any page may be printed without the unneeded sections and without colour.

While the website is currently not mobile-friendly, that is the next goal for improvement. Mobile devices are now so ubiquitous that if you do not design for mobile you are losing a large portion of your audience.

During my Web Development I class, I completed a simple newsletter signup form using JavaScript validation. I also researched issues to be aware of when developing a website, and possible website hosting solutions. Links to these pages:
